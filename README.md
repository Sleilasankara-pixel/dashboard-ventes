#  Dashboard Analyse des Ventes

Dashboard interactif d'analyse de ventes développé avec Python et Streamlit.

## Demo en ligne
[ Accéder au dashboard](https://dashboard-ventes-4cjgzqk33x96tll2nh6frz.streamlit.app/)

##  Fonctionnalités
- KPIs clés : chiffre d'affaires, profit, commandes, clients
- Filtres dynamiques par année, région et catégorie
- Visualisation des ventes par catégorie et région
- Évolution temporelle des ventes
- Top 10 des sous-catégories
- Tableau de données interactif

##  Stack technique
- Python
- Pandas
- Streamlit
- Matplotlib / Seaborn

##  Dataset
[Superstore Dataset - Kaggle](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)

##  Lancer le projet en local
pip install -r requirements.txt
streamlit run app.py
